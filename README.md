# Connect-4 Game

A JavaFX Connect-4 game built as an object-oriented programming project.

## Features

- JavaFX graphical interface
- Login and signup flow
- Player vs Player mode
- Player vs AI mode
- Easy and hard AI strategy options
- Score tracking
- File-based user storage
- Win and draw detection

## Technical Focus

- Java
- JavaFX
- Object-Oriented Programming
- Strategy Pattern
- Local file persistence
